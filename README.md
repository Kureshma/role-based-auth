# Role Based Authentication System

A simple role-based authentication web application built using **HTML, CSS, and JavaScript**.  
This project demonstrates how different users (Admin and User) can log in and access different pages based on their role.

---

##  Project Theme
**MoonPetal Accessories**

A soft, clean UI designed for an accessories brand, combined with authentication logic.

---

##  Features
- Admin Login
- User Login
- Role-based page access
- Separate dashboard/pages for Admin and User
- Clean and simple UI
- Beginner-friendly project structure

---

##  Technologies Used
- HTML
- CSS
- JavaScript

##  How to Run This Project
1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in any web browser
4. Click on login and test Admin/User flows
